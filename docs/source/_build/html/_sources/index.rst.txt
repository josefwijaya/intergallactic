Intergallactic - Gallactic JavaScript API
=========================================
Intergallactic is a collection of libraries which allow you to interact with
a local or remote ethereum node by using a HTTP connection

The following documentation will guide you through installing and running Intergallactic.
Besides that, This documentation also provide reference of API calls with examples.
